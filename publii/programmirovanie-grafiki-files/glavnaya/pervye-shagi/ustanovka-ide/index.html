Установка IDE

Главная :: Первые шаги :: Установка IDE

Можете установить либо Visual Studio, либо Rider

IDE это среда разработки включающая в себя как правило при установке весь набор необоходимых инструментов для программиста: комплиятор, отладчик, профилировщик и т.д.

Установка Visual Stduio:

1. https://visualstudio.microsoft.com/vs/community/

2. Нажмите кнопку Download (Скачать) и дождитесь загрузки файла

3. В загрузках найдите VisualStudioSetup.exe и запустите его

4. Убедитесь что выбран нужный нам пакет (.Net desktop development)

 

4. Нажимаем Install / Установить

5. Дождитесь завершения установки (на все вопросы отвечая Next / Далее)

6. Найдите в Пуске Visual Stduio и запустите её

7. Закрепите себе ее на панели задач для быстрого запуска в дальнейшем

 

Следующим шагом создадим первую программ

 

Если вы установили рускоязычную студию IDE, настоятельно рекомендую вам установить дополнительно Английский языковой пакет и переключить интерфейс на английский язык. В мире программирование все в основном происходит на английском языке (это не значит что вы не сможете создавать программы с русским интерфейсом). Но если у вас возникнут какие-то вопросы в ходе работы с IDE вы гораздо быстрее найдете ответы в интернете если будете пользоваться английскими терминами описывая свою ошибку или вопрос.
