Флаги

Главная :: Циклы :: Флаги

В предыдущем уроке мы познакомились как рисовать звезды и сделали для этого специальную отдельную функцию (пользоваться ей не обязательно, можете написать свое решение). Для закрепления навыков по рисованию различных фигур вы должны нарисовать серию флагов разных стран. Предлагаю вам сделать данное задание самостоятельно как хорошая практика для закрепления.

Флаг Арканзаса

Весьма комплексный флаг. Для вывод текста воспользуйтесь DrawString, для орисовки звезд по периметру ромба воспользуйтесь циклом или несколькими для каждой из сторон.

Флаг Колорадо

Для отрисовки неполного круга, можно либо создать функцию которая рисует неполный круг (с заданной в градусах разверткой), либо можно поверх полного круга нарисовать белый треугольник к центру. Возможно несколько вариантов. Поэкспеиментируейте 

Флаг Конфедерации 

Для отрисовки наклонных линий можно воспользоваться функцией DrawLine. Вам потребуется создать Pen с заданной толщиной .

Флаг США

Бело-красные полоски нужно выводить циклом. Звезды (так как это 2d сетка) нужно выводить двумя циклами (один вложен в другой). Для смещения строк воспользуйтесь признаком четности строки.

Флаг Кореи

Весьма сложный флаг, подумайте как можно его вывести. Сложность представляет волнистая линия в центре круга, а также прямоугольники повернутые относительно центры по краям круга. Для отрисовки волнистой линии можно воспользоваться синусоидой. Подумайте как применить клиппинг. Идеальное совпадение не требуется, главное чтобы это было близко к оригиналу. Для поворотов можно использовать матрицу поворота средств GDI, либо ручные повороты и смещения самих точек.
