Консоль: ввод / вывод

Для вывода текста на консоль используются функции Console.WriteLine , Console.Write

 

Для ввода текста с консоли исползуются функция Console.ReadLine и Console.Read

 

Переключение цветов выводимого текста. 

Часто текст выводимый в консоли удобно визуально маркировать разными цветами, например : красный - ошибка, желтый - предупреждение, зеленый - операция завершена успешно.
